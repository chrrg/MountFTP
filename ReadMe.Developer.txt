How to setup the development environment
	download and install the Dokan file system driver (http://dokan-dev.net/en/) 
		version 0.6.0 from http://dokan-dev.net/en/download/
		direct link: http://dokan-dev.net/wp-content/uploads/DokanInstall_0.6.0.exe
	make sure you have PowerShell 2 installed
	run the click_to_install_required_NuGet_packages.cmd in the solution folder
